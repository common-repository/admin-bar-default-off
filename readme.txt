This plugin sets new users to have the admin bar setting set off by default

== Description ==

By default WordPress creates users with their preferences set that the admin bar is enabled for them. This plugin sets the admin bar settings to off by default for new users. This plugin does not disable the admin bar globally.

== Installation ==

1. Upload `/admin-bar-default-off/admin-bar-default-off.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
